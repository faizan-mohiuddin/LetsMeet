Let's Meet: Home

Java


A powerful object oriented programming language that LetsMeet is written in. It was chosen for its familiarity with the team and its performance.

Spring Boot


The Spring framework was used to create LetsMeet's MVC (model-view-controller) architecture.

Google Maps/Calendar API


Thanks to Google, LetsMeet is able to provide its users with the ability to meet at a physical location and sync their events with their Google Calendars.

JavaScript


JavaScript is one of the most popular website development languages and was used to create a responsive and sophisticated front-end experience.

Bootstrap


Bootstrap is the popular open-source CSS framework used for creating responsive feeling neatly stylised websites.
